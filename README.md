# Tuya BYO 0.11.0

A Home Assistant custom integration for local Tuya / Smart Life devices using TinyTuya.

## Current focus
- Local control.
- Auto-created entities from Tuya mappings.
- Cleaner HomeKit export by exposing meaningful entities: fan, light, climate, switch, select, number and sensor.

## Security
Do not commit real local keys or Tuya secrets.
