# Tuya BYO

**Versión actual: 0.31.0** (debe coincidir con `custom_components/tuya_byo/manifest.json` — compruébalo tras cada actualización/HACS para confirmar que subió la versión correcta).

Local Tuya/Smart Life integration for Home Assistant using TinyTuya transport and Tuya Cloud knowledge.

## Current focus

- Stable HVAC control for Tuya/Midea/Johnson air conditioners.
- Correct ON/OFF semantics through the power DP.
- Local commands through TinyTuya.
